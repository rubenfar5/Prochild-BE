Crianças

Descrição

As crianças possuem a opção de escolher a faixa etária a que pertencem. Este cuidado existe para que a aplicação se adapte a cada faixa etária, de forma a ser mais intuitiva para a criança que a está a usar.
Esta aplicação é dedicada às crianças, com o intuito de terem uma plataforma para conhecerem os seus direitos e conseguirem ganhar mais voz na sociedade.

Funcionalidades

As funcionalidades, que serão detalhadas já de seguida, vão variar conforme a faixa etária. Mas de uma forma geral, as crianças podem:

  • Visualizar os seus direitos
  • Visualizar as linhas de apoio
  • Fazer denúncias
  • Utilizar um fórum
  • Acesso a diversões
